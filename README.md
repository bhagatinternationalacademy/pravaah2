# PRAVAAH ERP

PRAVAAH (Integrated Management Suite) is a centralized ERP platform developed as part of the L&T Full Stack Development Program.

## Modules

* User Management
* Participant Management
* Faculty Management
* Course Management
* Hostel Management
* Common Modules
* Dashboard & Reports

## Technology Stack

* Django
* MySQL
* HTML
* CSS
* JavaScript
* Bootstrap 5

## Important Documents

Before starting development, every team member must read:

1. UI_STYLE_GUIDE.md
2. INTEGRATION_GUIDELINES.md

## Team Development Rules

* Work only inside your assigned module.
* Do not modify shared project files.
* Follow the approved PRAVAAH UI design.
* Use Git branches for development.
* Submit changes through Pull Requests.

## Repository Structure

Each module is developed independently and integrated into a single Django application.

All integration decisions are handled by the User Management / Integration Team.
